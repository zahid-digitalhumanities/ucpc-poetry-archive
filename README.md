# UCPC Poetry Archive

A research-grade bilingual digital humanities platform for Urdu ghazals and nazms.

## Features
- Bilingual content (Urdu/English)
- Advanced search (Roman Urdu, fuzzy, voice input)
- Bulk upload with duplicate detection
- Poet-centric browsing
- Modern responsive UI

## Installation
1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Set up PostgreSQL and update `config.py` with your database URL.
6. Run: `python app.py`
